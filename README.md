# SuperSMP Companion App

An Android Companion app for the SuperSMP Minecraft server.

# SuperSMP Companion App Features
Current Features are:

> In-App-Voting, In-App-Shop (With the Official SuperSMP Shop), In-App-Map (With the Official SuperSMP Map)

# SuperSMP Companion App Features Planned
Planned Features are:

> Migrating to Kotlin, Jetpack Compose and Kotlin DSL
> Adding Screenshots and Videos to YouTube or other Providers for showing how the App Works.
